# Overview & Introduction

## What is ImaginaryWidget?

ImaginaryWidget is a lightweight, embeddable JavaScript widget that provides seamless file upload functionality for web applications. Built with vanilla JavaScript, it offers a modern, user-friendly interface for uploading files to your ImaginaryStore backend.

The widget handles all the complexity of file uploads, including:
- Progress tracking
- File validation
- Error handling
- Token-based authentication
- Drag and drop support
