# Quick Start Guide

Get ImaginaryWidget up and running in your web application in just a few minutes.

## Prerequisites

Before you begin, ensure you have:

- ‚úÖ A modern web browser (Chrome 60+, Firefox 55+, Safari 12+, Edge 79+)
- ‚úÖ An ImaginaryStore backend API running
- ‚úÖ A valid API key from your ImaginaryStore dashboard
- ‚úÖ Basic knowledge of HTML and JavaScript

## Basic Installation Steps

### Step 1: Include the Widget Script

Add the ImaginaryWidget script to your HTML page:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My App with File Upload</title>
</head>
<body>
    <!-- Your page content -->
    <button id="upload-btn">Upload Files</button>
    
    <!-- Include ImaginaryWidget -->
    <script src="https://cdn.imaginarystore.com/widget/v1/widget.js"></script>
</body>
</html>
```

### Step 2: Initialize the Widget

Create a new widget instance and configure it:

```javascript
// Initialize the widget
const widget = new ImaginaryWidget();

// Configure and initialize
widget.init({
    apiKey: 'your-public-api-key-here',
    apiUrl: 'https://your-backend.com/api',
    onSuccess: (files) => {
        console.log('Upload successful:', files);
        alert('Files uploaded successfully!');
    },
    onError: (error) => {
        console.error('Upload failed:', error);
        alert('Upload failed: ' + error.message);
    }
});
```

### Step 3: Add Upload Trigger

Connect the widget to a button or event:

```javascript
// Get the upload button
const uploadButton = document.getElementById('upload-btn');

// Open widget when button is clicked
uploadButton.addEventListener('click', () => {
    widget.open();
});
```
